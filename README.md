## What is SwarmCrafter?

SwarmCrafter is a mobile app that lets you manage your Minecraft accounts and run automations — straight from your Android phone.

The core of the app is its **automation system**. You build flows: sequences of actions your bots carry out automatically. Farm, grind, stay active, react to events — you design the logic visually, no coding needed. Set it up once and let it run in the background while you get on with your day.

It's built to be powerful without being complicated. Whether you're running a single account or managing several, everything lives in one clean interface on your phone.

> **Currently in closed beta.** Access is invite-only while we test and refine the experience.

## Download

Head to the [**Releases**](../../releases) tab and grab the latest `.apk`.

### Installing the APK

1. Download the `.apk` on your Android device
2. If prompted, allow installation from unknown sources in your settings
3. Open the file to install
4. Launch SwarmCrafter and enter your access code to unlock the app

> Requires Android 8.0 or later.
