## What is SwarmCrafter?

SwarmCrafter lets you manage and automate your Minecraft accounts directly from your Android phone. Set up tasks, keep accounts active, and run automations — all from a clean, modern mobile app. No PC, no always-on computer, just your phone.

> **Currently in closed beta.** Access is invite-only while we test and refine the experience.

## Download

Head to the [**Releases**](../../releases) tab and grab the latest `.apk`.

### Installing the APK

1. Download the `.apk` file on your Android device.
2. If prompted, allow installation from unknown sources in your settings.
3. Open the file to install.
4. Launch SwarmCrafter and enter your access code to get started.

> **Requires Android 8.0 or later.**

## Getting Access

SwarmCrafter is currently invite-only. If you've received an access code, enter it on the first launch screen to unlock the app.

Interested in joining the beta? Reach out to get on the waitlist.
